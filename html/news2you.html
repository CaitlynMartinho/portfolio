News2you WordPress Website

Project Overview

Project Type: Website to WordPress Redesign

Role: UI/UX Designer, WordPress Developer

Tools Used: Figma, Xampp, WordPress, Photoshop

Languages Used: CSS

1. Project Overview

As part of a college project, I redesigned the News2You website using WordPress to improve its visual appeal, accessibility, and user engagement. The new design focuses on inclusive education, modern layout structure, and easy navigation. It highlights key offerings like educational resources and teaching tools while ensuring a responsive and user-friendly experience across devices.
